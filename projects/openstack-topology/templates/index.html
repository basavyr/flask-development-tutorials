Topology view

Shows a topological view with all the nodes available on the Openstack infrastructure. The node types are organized hierarchically, with the top priority being for the controller nodes.

The information for each node is pulled from a database that is created/updated before the webpage is loaded.

Histogram view

Draws a histogram with all the node types available on the Openstack infrastructure. There are five types of nodes in the current Openstack cluster.

  • controller-nodes
  • compute-nodes
  • conductor-nodes
  • scheduler-nodes
  • consoleauth-nodes

For each type, the total number of nodes is graphically represented.

Graph view

This service shows system information and statistics for the main node on which the Openstack cluster is deployed. Data is generated when the webpage is accessed.
